Должно быть так: Управление игрой, присоединиться к игре, создание игры, отмена(Для скрытия клавиатуры)
    В управлении должна быть функция изменения, просмотра присоединившихся, начать игру, удалить игру

1. Пользователю предлагается запустить игру, редактировать игру , присоединиться к игре, создать игру, удалить игру и отменить(убрать ввод клавиатуры). У игры есть уникальный идентификатор
    1.1 Пользователь выбирает создать игру
            ОДНА ТАБЛИЦА
        - заносим в базу его айди
        - генерируем уникальный хэш(или создатель придумывает уникальное название)
        - указываем количество человек, которое будет играть

            Дальше создателю возвращается хэш и он должен отправить его своим друзьям. Далее друзья присоединяются к игре

            ДРУГАЯ ТАБЛИЦА
        - храним айди первой таблицы
        - айдишки пользователей, которые присоединились к игре(не должны дублироваться)
    1.2. Пользователь выбирает редактировать игру
        - может изменить количество участников игры
    1.3 Пользователь выбирает присоединиться к игре
        - пользователь должен ввести уникальный хэш
            - хэш не верный
            - пользователь уже состоит в игре
        - создатель не может еще раз присоединиться
        - пользователь должен ввести свое имя
    1.4 Пользователь выбирает запустить игру
        - проверка на то, что указанное количество пользователей, соответствует зарегистрированным
        - для каждого пользователя нужно подобрать человека, которому он будет дарить подарок (просто проходиться циклом по каждому пользователю и шафлом вытаскивать другого.)
            - просто проходиться циклом по каждому пользователю и шафлом вытаскивать другого. Если пользователь не он сам, то убирать из всех участников двух игроков.
        - после того, как игра запустилась, она не может быть отредактирована
    1.5 Пользователь выбирает удалить игру
        - нужно для того, что если он создал игру, но потом они передумали. Чтобы не приходили сообщения
    1.6 Пользователь нажимает кнопку отменить
        - убирается клавиатура
